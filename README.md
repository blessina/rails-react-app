Tracks Api http://localhost:3000/api/v1/tracks
Localhost: http://localhost:5173/

* Check out the code https://github.com/blessina/rails-react-app

* Install the specified ruby and rails versions

* Run bundle install & npm install

* Seed the database with the fake track data. rails db:seed

* rails s to start the server & npm run dev to start the react server

* Browse to http://localhost:5173/ to view the tracks
